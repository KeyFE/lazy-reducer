# lazy-reducer
Dynamically load reducer.

## Install
```
npm install --save lazy-reducer
```
## API

#### `lazyReducerEnhancer`

#### `<LazyReducer />`

#### `withLazyReducer`

```js
callback(err, {
    [key]: asyncReducer
});
```

#### `lazyReducerEnhancer(syncReducer)`
redux enhancer
